Ex00: Markdown Cheatsheet.

1. Encabezados (Headers)

MarkdownResultado (HTML)
# H1

Título 1

## H2

Título 2

### H3

Título 3

#### H4

Título 4

2. Énfasis de Texto

**Negrita** → Negrita
*Cursiva* o _Cursiva_ → Cursiva
***Negrita y Cursiva*** → Ambas
~~Tachado~~ → Tachado

3. Listas

Desordenadas: 
- Elemento 1
- Elemento 2
  - Sub-elemento
Ordenadas: 
1. Primero
2. Segundo
3. Tercero

4. Enlaces e Imágenes

TipoSintaxis
Enlace[Texto](https://url.com)
Imagen![Alt text](url_imagen.jpg)
Referencia[Texto][id] ... [id]: url

5. Bloques de Código

Código en línea: `print("hola")`

Bloque de código (con lenguaje para resaltado):

```python
def saludo():
    print("Hola Mundo")
```

7. Otros Elementos

  • Citas: > Este es un bloque de cita.
  • Línea horizontal: --- o ***
  • Task Lists: - [x] Tarea hecha | - [ ] Pendiente
