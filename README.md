# Django Piscine

## Project Overview

Welcome to the **Django Piscine** repository! This project is part of a learning journey focused on mastering Django, a high-level Python web framework. Django is designed to encourage rapid development and clean, pragmatic design, making it an excellent tool for building robust and scalable web applications.

The repository contains exercises and projects that progressively introduce key concepts of Django development, including models, views, templates, forms, and database integration. It also explores advanced topics such as middleware, authentication, and deployment.

## Structure

The repository is organized into modules (`m00`, `m01`, `m02`, `m03`), each containing directories (`d00`, `d01`, etc.) with exercises (`ex00`, `ex01`, etc.). Here's an overview of the structure:

### 1. **Exercises**
   - **Basic Django Setup**: Learn how to set up a Django project and application.
   - **HTML and CSS Integration**: Explore how to integrate static files like HTML, CSS, and JavaScript into Django projects.
   - **Forms and Validation**: Implement forms and handle user input securely.
   - **Database Models**: Define models and interact with databases using Django's ORM.
   - **Views and Templates**: Create dynamic views and render templates.
   - **Advanced Topics**: Dive into middleware, authentication, and deployment.

### 2. **Supporting Files**
   - **`requirements.txt`**: Lists dependencies required for the project.
   - **`.gitignore`**: Specifies files and directories to be ignored by Git.
   - **`README.md`**: Documentation for the project.

## Competencies Developed

By working on this project, participants will develop the following skills:

- **Web Development**: Build web applications using Django and integrate frontend technologies like HTML, CSS, and JavaScript.
- **Python Programming**: Write Python scripts to define models, views, and templates.
- **Database Management**: Use Django's ORM to interact with databases and manage data efficiently.
- **Problem-Solving**: Debug and optimize code to ensure functionality and scalability.
- **Deployment**: Learn how to deploy Django applications to production environments.

## How to Use This Repository

1. Clone the repository to your local machine:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the relevant module and exercise directory to begin solving challenges.
3. Install dependencies using `pip`:
   ```bash
   pip install -r requirements.txt
   ```
4. Run the Django development server:
   ```bash
   python manage.py runserver
   ```

## Notes

- Ensure you have Python and Django installed on your system.
- Follow the instructions provided in each exercise to complete the tasks.

## Conclusion

The **Django Piscine** is an excellent opportunity to learn Django and web development in a structured and hands-on manner. Whether you're new to Django or looking to deepen your knowledge, this repository provides a comprehensive learning experience.

Happy coding!
